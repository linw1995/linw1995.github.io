比较简单的搭梯子方法有两个：

  1. Lantern 蓝灯
  2. Shadowsocks 影梭

Lantern 蓝灯

下载

蓝灯是个开源项目，在程序员交友平台GitHub上就能下载。
传送门：

  • github
  • lantern

使用方法

非常简单，只需要打开该软件就好。
虽说免费用户有800m的月流量，但网速不够支持看视频；
专业用户虽然一年很便宜，但不适合一些不经常用梯子的人。

Shadowsocks 影梭

下载

  • github-wiki

使用方法

  1. 下载解压后即可打开使用
  2. 在任务栏中找到图标，右键打开菜单，选中启用系统代理
  3. 选用适合的服务器
    完成第二步后还没办法看墙外的风景，需要添加一些代理服务器。
    那么如何添加代理服务器呢？
    有两种方式：
    • 网上搜索一些免费的代理服务器，但这些免费的代理服务器不仅慢还有安全隐患。
    • 去正规的服务器运营商sscloud购买这类服务
  4. 添加代理服务器
    双击任务栏中的图标，弹出以下窗口 手动填入信息
    或者扫描服务器运营商提供的二维码
  5. 完成以上步骤，就可以在系统代理模式==》全局代理模式愉快地观赏墙外的世界

PCA模式

全局代理模式下，上墙内网站也会使用你的服务器流量。这就需要使用系统代理模式==》PAC模式来智能代理。

系统代理模式中选择PAC代理，然后在PAC中选择从GFWList更新本地PAC。
这样就可以智能地搭梯子了。

什么！你说你浏览器还是打不开墙外的世界

原因：

  1. 在绝大数浏览器设置里都有一项选择代理方式地选项，只要更改为使用系统代理设置，就可以解决问题。
  2. 有些浏览器像FireFox、Chrome等，都需要安装代理插件才能使用系统代理。

FireFox浏览器的配置方法

  1. 首先安装代理组件FoxyProxy
  2. 新建或编辑已有的代理服务器选项
  3. 添加本地PAC文件的位置，点击测试看是否通过
  4. 通过后，修改下PAC文件
  5. 重新测试PAC文件是否通过
  6. 现在可以试着打开墙外的世界 一个旋转的FoxyProxy图标，和一个正在打开的墙外世界。

Chrome浏览器的配置方法

  1. 首先安装Proxy SwitchyOmega

  2. 进入SwitchyOmega 选项·情景模式中的proxy添加代理服务器

  3. 自定需代理的网站，点击浏览器右上角的小图标

  4. 也可以开启全局模式，把auto switch改成proxy即可

注意

  • 可以在Shadowsocks服务器选项中选择不同的服务器来满足不同的需求，平时选择高可用 就行
  • 最后，谨慎了解墙外的信息
