函数 py2 py3 描述
cmp_to_key ✔ ✔ Transform an old-style comparison function to a key function.
total_ordering ✔ ✔ Given a class defining one or more rich comparison ordering methods, this class decorator supplies the rest.
reduce ✔ ✔ This is the same function as reduce().
partial ✔ ✔ Return a new partial object which when called will behave like func called with the positional arguments args and keyword arguments keywords.

lru_cache

Least Recently Used(LRU)

装饰器，用来缓存函数对应不同参数的返回结果。

可以提供maxsize=None与typed=False这两个参数，
参数maxsize为最多缓存的次数，如果为None，则无限制，设置为2n时，性能最佳，特别适合于耗时的函数。
typed=True，则不同参数类型的调用将分别缓存，例如 f(3) 和 f(3.0)。

被装饰的函数会有cache_clear和cache_info这两个方法，分别用于清除缓存和查看缓存信息
